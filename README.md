# Simpleplot

A simple plot liblary used to plot graph with arrtibuted vertices and edges.

## Usage

### Vectice' Attribute
- color

### Edge's Attribute
- color
- direction
- linked vertices

## Example


```cmd
python simpleplot --edges edges.txt --output simpleplot.html
```
